Sámedikki nuoraidráđđi joatká guovtte mearreáigásaš ofelačča ohcanáigge 14.11.2014 rádjai. Sääʹmteeʹǧǧ nuõrisuåvtõs juätkk kueiʹt mieʹrräiggsaž ofelaž ooccâmääiʹj 14.11.2014 räjja.
