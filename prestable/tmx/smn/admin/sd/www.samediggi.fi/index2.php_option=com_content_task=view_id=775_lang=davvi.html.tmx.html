Bovdehus sámenuoraid konferensii Povdiittâs säminuorâi konfereensân
Buresboahtin sámenuoraid konferensii sámekulturguovddáš Sajosii Anárii 6.-7.2.2014. Konfereanssa lágida Suoma Sámedikki nuoraidráđđi. Tiervâpuáttim Sämitige nuorâirääđi uárnim säminuorâi konfereensân säämi kulttuurkuávdáš Sajosân Anarân 6.-7.2.2014.
Konfereanssa fáddán lea utnolašvuohta ja sámenuoraid identitehta. Konferens fáddán lii soováášvuotâ sehe säminuorâi identiteet.
Konfereanssa eahketprográmmas lea sámemusihkka. Konferens eehidohjelmin lii sämimuusik.
Konfereansa lea nuvttá ja dat lea rabas buohkaide geat leat beroštuvvan. Konferens lii mávsuttem já tot lii áávus puohháid kiäh láá perustum.
Almmuheamit maŋimustá 24.1.2014. Almottâtmeh majemustáá 24.1.2014.
