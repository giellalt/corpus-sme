Sopâmuš valmâštâllâmmuddoost lâi kuittâg oovtčielgiitlávt čielgâs, ete artikkâl 8(j) kuáská algâ- já hiäimuaalmugáid, veikâ tekstâ ij tagarin šoddâmgin sopâmušân.
