Almmuhanáigi Sámenuoraid dáiddadáhpáhussii álgán Almottâttâm Säminuorâi taaidâtábátusân lii álgám
Almmuhanáigi Sámenuoraid dáiddadáhpáhussii, mii ordnejuvvo Vuohčus njukčamánu 26. beaivve, lea álgán. Almottâttâm Säminuorâi taaidâtábátusân, mii uárnejuvvoo Vuáčust njuhčâmáánu 26. peeivi, lii álgám.
Dáhpáhusa temán leat teáhter ja oanehisfilbma. Tábáhtus teeman lii teatter já uánihiselleekove.
Almmuhanáigi teáhtergilvui nohká 2.3. ja filbmafestiválii 13.3. Almottâttâm teatterkištodmân nohá 2.3. já elleekovefestivaalân 13.3.
Dárkilut dieđut dáhpáhussii almmuheamis dás: Tärhibeh tiäđuh almottâtmist täst:
Dáiddadáhpáhusa 2009 njuolggadusat já
