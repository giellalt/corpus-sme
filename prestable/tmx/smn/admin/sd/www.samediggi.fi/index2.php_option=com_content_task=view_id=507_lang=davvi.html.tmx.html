Davviriikkalaš álbmotdearvvasvuođakonfereansa Turkus 24.8.2011 Tave-eennâmlâš aalmugtiervâsvuođâkonferens Tuurkust 24.8.2011
Mat leat davviriikkalaš buresbirgenriikka stuorámus hástalusat sámeperspektiivvas ? Moh láá tave-eennâmli pyereestváijeemstaatâ stuárráámusah hástuseh sämmilâšuáinimčievâst ?
Ášši gieđahallo 24.8.2011 davviriikkalaš álbmotdearvvasvuođakonfereanssas Turkus. Ääši kieđâvušeh 24.8.2011 teve-eennâlmlii aalmugtiervâsvuođâkonfereensâst Tuurkust.
Fáddái laktáseaddji ovdaságat fállet refleksiivvalaš sámeperspektiivvaid sihke sosiála- ja dearvvasvuođabálvalusaid buvttadeapmái, dearvvasvuođaovddideapmái ja sámiid dearvvasvuođadillái guoski guovddáš gažaldagaide deattuhemiin erenomážit kultuvrralaš perspektiivva. Fáádán lohtâseijee ovdâsaavah fäällih refleksiivlijd sämmilâšuáinimčievâid sehe sosiaal- já tiervâsvuođâpalvâlusâi pyevtitmijd, tiervâsvuođâovdedem já sämmilij tiervâsvuođâtile kyeskee kuávdášlâš koččâmâšasâttâlmáid tiäddutmáin eromâšávt kulttuurlii perspektiiv.
