Videostudio
Alladását videostudio heive mánggakámeravuogádaga atnui ja earáge videomateriála govvemii. Ollâtásásâš videostudio heivee maaŋgâkameravuáháduv kiävtun já eres-uv videomateriaal kuvviimân.
Studiolanjaide gullet studio, live-editerenlatnja ja editerenlanjat. Studiotiilijd kullojeh studio, live-editistemtile sehe editistemtileh.
Studiolanjain vástida Sámi oahpahusguovddáš, muhto guovddáža olggobealvárrejumit bohtet Sajosa Kongreassa- ja dáhpáhusbálvalusaid bokte. Studiotiilijn västid Säämi máttááttâskuávdáš, mutâ kuávdáá olgopiälásiih väärridmeh puátih Saijoos Kongres- já tábáhtuspalvâlusâi peht.
Studio lea Sajosa vuosttaš gearddis, váldouvssa gurutbealde Sámi oahpahusguovddáža media- ja giellaluohkáid oktavuođas. Studio lii Saijoos vuossâmuu keerdist, váldu-uuvsâ čižetpeln Säämi máttááttâskuávdáá media- já kielâmáttááttâstiilij ohtâvuođâst.
