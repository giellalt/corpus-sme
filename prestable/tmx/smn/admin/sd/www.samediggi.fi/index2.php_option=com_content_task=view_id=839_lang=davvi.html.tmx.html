Ijahis idja lágiduvvo borgemánus oktanuppelogát geardde Ijahis idja uárnejuvvoo porgemáánust ohtnubálovváád keerdi
Jagi 2014 eamiálbmogiid musihkkadáhpáhus Ijahis idja lágiduvvo dasa stáđásmuvvan áigge borgemánu gaskka muttus Anáris, Sámekulturguovddáš Sajosis. Ive 2014 algâaalmugij muusiktábáhtus Ijâttes ijjâ uárnejuvvoo toos vuáhádum ääigi porgemáánu pelimuddoost Anarist, Säämi kulttuurkuávdáš Sajosist.
Festivála dáhpáhusbeaivvit leat 15–17.8.2014 ja dáhpáhusa dán jagaš temá lea “ Jávrebiekkat ”. Festivaal tábáhtuspeeivih láá 15–17.8.2014 já tábáhtus taanivvááš teema lii “ Jávrebiekkat ”.
Oktanuppelogát geardde lágiduvvon dáhpáhusas bajiduvvojit ovdan smávva sámi vehátlogut dahje nuortalaččat ja anáraččat sihke sin musihkka- ja dánsaárbi, dego nuortalaškatrilla, livđe ja leuŠdd. Tábáhtusâst, mii uárnejuvvoo ohtnubálovváád keerdi, pajedeh oovdân uccâ sämiucceeblovoid ađai nuorttâlâšâid já anarâšâid sehe sii muusik- já tanssâärbivuovijd, tego nuorttâlâškatrilli, livđe já leuŠdd.
Ijahis idja lea áidna sámemusihkkafestivála oppa Suomas ja das lea gulul šaddan okta dehálamos Davvi-Sámi geassedáhpáhusain. Ijâttes ijjâ lii áinoo sämimuusikfestivaal ubâ Suomâst já tast lii kuuloold šoddâm ohtâ puoh merhâšittee Tave-Laapi kesitábáhtus.
Diibmá logijagaš mátkki ávvudan festiválii sihke dan siidodáhpáhusaide oassálaste guovtti beaivvi áigge oktiibuot 2800 fidni. Moonnâm ive loveivvááš juuhlijd toollâm festivaalân sehe ton miäldustábáhtussáid uásálistij kyevti peeivi ääigi ulâttâsmeeri, ohtsis 2800 olmožid.
Jagi 2014 Ijahis ija prográmmii gullet nu árbevirolaš sámemusihka čehpiid loaiddasteamit, šaddi sámeartisttaid konsearttat, mánáid prográmma, eamiálbmotgussiid musihkkaloaiddasteamit sihke sámemusihkka klubbaeahkedat. Ive 2014 Ijâttes iijâ ohjelmân kuleh nuuvt ärbivuáválii sämimuusik taiđârij lávdástmeh, pajaneijee sämiartistij konserteh, párnái ohjelm, algâaalmugkuosij muusik sehe sämimuusikân vuáijoo klubiehideh.
Okta festivála konsearttain lágiduvvo Sajosa auditorias, muhto muđuid dáhpáhus ollašuhttojuvvo árbevirolaš vuogi mielde olgoáibmofestiválan. Ohtâ festivaal konsertijn uárnejuvvoo Saijoos auditoriost, mutâ mudoi tábáhtus olášuttoo olgon, tego táválávt.
