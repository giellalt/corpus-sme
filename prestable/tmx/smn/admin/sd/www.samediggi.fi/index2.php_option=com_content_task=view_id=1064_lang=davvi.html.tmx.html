Sámediggi attii stipeanddaid Sámi oahpahusguovddážis válmmaštuvvan oahppiide Sämitigge adelij stipendijd Säämi máttááttâskuávdáást valmâštum uáppeid
Sámediggi attii stipeandda njealji viššalis sámegiela oahppái, geat válmmaštuvve Sámi oahpahusguovddážis. Sämitigge adelij stiipeend neelji iäljáris sämikielâ uáppei, kiäh valmâštuvvii Säämi máttááttâskuávdáást.
Sámi oahpahusguovddáža giđđafeasta dollojuvvui gaskavahku geassemánu goalmmát beaivve. Säämi máttááttâskuávdáá kiđđâjuhleh onnii koskoho kesimáánu kuálmád peeivi.
