Buđaldusteáhter Ivaloiset Sajosis Puđâldâsteatter Ivaloiset Sajosist
Anárlaš buđaldusteáhter Ivaloiset čájeha Aimo Vuorinen čállán čájálmasa Votkaturistit Sajosa Dolla-sáles duorastaga 2.2.2012 dmu 19. Aanaarlâš puđâldâsteatter Ivaloiset oovdânpuáhtá Aimo Vuoris čáállám čáitálduv Votkaturistit Sajos Dolla-Salist tuorâstuv 2.2.2012 tme 19.
Bileahtat uvssa luhtte dmu 18 rájes. Liipuid uuvsâst tme 18 rääjist.
Bagadalli: Ritva Savela. Stivrejeijee: Ritva Savela.
