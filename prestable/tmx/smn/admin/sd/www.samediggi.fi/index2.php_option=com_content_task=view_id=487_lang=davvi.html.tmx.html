Ella Näkkäläjärvi (2011)
Lean 17-jahkásaš ja ásan Eanodaga Neakkelis. Ella Näkkäläjärvi (2011) Mun lam 17-ihásâš já aasâm Iänuduv Näkkälist.
Stuđeren Eanodagas logahagas vuosttas jagi. Jođám Iänuduv luvâttuv vuossâmuu ive.
Láven lohkat, lihkadit olgun ja muhtun veardde duddjot. Lijkkuum lihâdiđ olgon, luuvâm já moottoom verd tuoijuum.
Áhččán lea badjealmmái, nu ahte mii oažžut áigáiboađumet boazodoalus. Iäččám lii pajeolmooš já eellip puásuituálust.
Mun lean beroštuvvan sápmelašvuođas ja sámekultuvrras. Tuubdâm čuuvtij mielâkiddiivâšvuođâ sämmilâšvuotân já ton kulttuuráid.
Lean álo mielde miessemearkumiin ja dálverátkimiin. Mun lam ovttuu mieldi vyesimiärkkumijn já täviv pygálusâin.
Mis leat bohccot ruoktošiljus duollet dálle. Mist láá päikkipoccuuh ain motomin.
Mii eat hála ruovttus sámegiela, muhto áddet muhtun veardde. Mii päikkiulmuuh iä sáárnu sämikielâ, mut iberdeh kale mottoomverd.
Álgen sámedikki nuoraidráđđái danin go lean beroštuvvan áššis.
Gaskkat leat goit guhkit iige mus leat vel vuodjinkoarta. Servim sämitige nuorâirááđán mielâkiddiivâšvuođâ keežild, mut jotteemmääđhih láá kuheh, ege must ij lah vala vyeijimkorttâ.
