Anna Näkkäläjärvist vs. kulttuurčällee
Sámedikki almmolaš doaimmahagas ohcanláhkai leamašan kulturčálli virgesadjásažžan lea váldojuvvon musihka magisttar Anna Näkkäläjärvi. Sämitige almos toimâttuvâst uuccâmnáálá lamaš kulttuurčällee virgesajasâžžân lii valdum muusik maister Anna Näkkäläjärvi.
Näkkäläjärvi álggaha barggustis miessemánu 18. beaivve 2009. Näkkäläjärvi aalgât pargostis 18. vyesimáánu 2009.
Su doaibmabáiki lea Sámedikki almmolaš doaimmahagas Anára Rivdolis. Suu toimâsaje lii Sämitige almos toimâttuvâst Aanaar Rivdulist.
