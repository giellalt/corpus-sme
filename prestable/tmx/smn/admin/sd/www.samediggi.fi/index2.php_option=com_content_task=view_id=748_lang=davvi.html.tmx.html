Sámenuoraid dáiddadáhpáhus 2014 ordnejuvvo Soađegilis Säminuorâi taaiđâtábáhtus 2014 uárnejuvvoo Suáđigilist
Sámenuoraid dáiddadáhpáhus 2014 ordnejuvvo Soađegili girkosiiddas. Säminuorâi taaiđâtábáhtus 2014 uárnejuvvoo Suáđigil markkânist.
Dáhpáhusbáikin boahtá leahkit Soađegili ođđa valáštallanhálla girkosiidda guovddážis. Tábáhtussaijeen lii Suáđigil uđđâ lihâdemhallâ markkân kuávdáást.
Dáiddadáhpáhusa riikka dási temán lea dánsun. Taaiđâtábáhtus väldikodálâš teema lii tanssâ.
Sámediggi ordne dáhpáhusa ovttasbarggus Soađegili gielddain. Tábáhtus uárnejuvvoo sämitige já Suáđigil kieldâ oovtâstpargon.
Sámedikki skuvlen- ja oahppamateriálalávdegoddi oahpásnuvai ođđa valáštallanhállii ja mearridii dáhpáhusa ordnemis Soađegilis čoahkkimistis čakčamánu 23. beaivve. Sämitige škovlim- já oppâmateriaallävdikodde uápásmui uđđâ lihâdemhaalân já meridij tábáhtus orniimist Suáđigilist čuákkimistis 23. čohčâmáánu.
Dáhpáhus ordnejuvvo juogo vahkus 12 dahje 13. Tábáhtus uárnejuvvoo jo-uv ohhoost 12 teikâ 13.
Lávdegoddi mearrida dáhpáhusa beaivemearis čuovvovaš čoahkkimistis golggotmánu loahpas. Lävdikodde meerrid táháhtus peivimere čuávvoo čuákkimist roovvâdmáánu loopâst.
Dáhpáhusa riikka dási temás, mii lea dánsun, leat gávdnamis dárkilut dieđut siidduin http://sottiisimoves.fi/saannot_sa /. Tábáhtus välkikodálii teemast, tanssâst, láá tärhibeh tiäđuh siijđoin http://sottiisimoves.fi/saannot_sa /.
Dáhpáhussii árbevirolaččat gullevaš sámi kultuvrii laktáseaddji temás ii leat vel mearriduvvon. Tábáhtusân ärbivuáválávt kullee sämikulttuurân lohtâseijee šlaajâst iä lah vala rahtum miärádâsah.
Skuvlen- ja oahppamateriáladoaimmahat váldá vuostá evttohusaid, maid sáhttá dahkat golggotmánu 25. beaivvi rádjái. Škvolim- já oppâmateriaaltoimâttâh váldá vuástá iävtuttâsâid, maid puáhtá toohâđ sleđgâpostáin roovvâdmáánu 25. peeivi räi.
Lassedieđut dáiddadáhpáhusas dás. Lasetiäđuh taaiđâtábáhtusâst täst.
