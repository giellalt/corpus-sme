Sámenuoraid rock-joavkkut menestuvve Turku dáhpáhusas Säminuorâi muusikjuávhuh miänástuvvii Tuurku tábáhtusâst
Sámenuoraid ovddasteaddjit riikka dási Sounds-dáhpáhusas, anárlaš The Slap ja ohcejohkalaš Hilbes, ožžo giitosiid loaiddastemiin ja musihkalaš máhtuin. Aanaarlâš The Slap já ucjuvlâš Hilbes, moh ovdâstii Säminuorâid väldikodálâš Sounds 2012 - tábáhtusâst, ožžuu kijttosijd lávdástâlmijnis já mättimijnis.
Goabbáge joavku oaččui gudnemáinnašumi Turku-dáhpáhusas. Kuohtuuh juávhuh ožžuu tábáhtusâst kunneemainâšume.
The Slap ja Hilbes válljejuvvuiga sámenuoraid ovddasteaddjin riikka dási dáhpáhussii Anáris njukčamánu loahpas dollojuvvon Sámenuoraid dáiddadáhpáhusas. The Slap já Hilbes väljejuin Anarist njuhčâmáánu loopâst uárnejum Säminuorâi taaidâtábáhtusâst säminuorâi ovdâsteijen väldikodálâš tábáhtusân.
Loga Sounds 2012-dáhpáhusdoaimmahusa čállosa Hilbes-joavkkus dás. Luuvâ Sounds 2012-tábáhtustoimâttâs čalluu Hilbes-juávhust täst.
Loga ođđasa Anáris dollojuvvon Sámenuoraid dáiddadáhpáhusas dás. Luuvâ uđđâs Anarist toollum Säminuorâi tááidâtábáhtusâst täst.
