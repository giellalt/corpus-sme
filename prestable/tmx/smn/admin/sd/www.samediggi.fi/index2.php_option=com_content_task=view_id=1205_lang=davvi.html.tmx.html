Tiina Sanila-Aikio joatká Sámedikki ságadoallin - várreságadoallin Heikki Paltto ja Tuomas Aslak Juuso Tiina Sanila-Aikio juátká sämitige saavâjođetteijen - värisaavâjođetteijen Heikki Paltto já Tuomas Aslak Juuso
Tiina Sanila-Aikio joatká Sámedikki ságadoallin válgabaji 2016-2019. Tiina Sanila-Aikio juátká sämitige saavâjođetteijen vaaljâpaje 2016–2019.
Vuosttaš várreságadoallin joatká Heikki Paltto Anáris. Vuossâmuš värisaavâjođetteijen juátká Heikki Paltto Anarist.
Nubbin várreságadoallin álggaha Tuomas Aslak Juuso Eanodagas. Nubben saavâjođetteijen aalgât Tuomas Aslak Juuso Iänuduvvâst.
Sámediggi čoahkkanii válgabaji vuosttaš čoahkkimii Anáris Sámekulturguovddáš Sajosis 23.-24.2. Sämitigge čokkânij vaaljâpaje vuossâmuu čuákkimân Aanaar Sajosân 23.–24.2.
