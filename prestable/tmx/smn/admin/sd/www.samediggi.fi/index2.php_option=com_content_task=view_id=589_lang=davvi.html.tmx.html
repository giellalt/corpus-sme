Sajosa rahpan 2012 Saijoos lekkâmeh 2012
Govat Sajosa vihaheamis ja Sámedikki rahpamis ja kultureahkedis 3.4.2012. Koveh Saijoos vihkjáin já Sämitige lekkâmijn sehe kulttuurehidist 3.4.2012.
Lassin dáppe gávdnojit govat Sajosa rahpamii guoskevaš Álbmotfeasttas / Rabas uvssaid beaivvis, mii dollojuvvui 4.4.2012. Lasseen taan vuárhást láá koveh Saijoos lekkâmáid lohtâseijee Aalmugjuhleest / Ávus uuvsâin, mii tollui 4.4.2012.
Dásseválddi Presideantta sáhka Sajosa vihaheamis ja Sámedikki rahpamis 3.4.2012. Täsivääldi President saahâ Saijoos vihkjáin já Sämitige lekkâmijn 3.4.2012.
Dásseválddi Presideanta ja roavvá Jenni Haukio boahtiba Sajosa sálii. Täsivääldi President já rovvá Jenni Haukio puáttiv Saijoos salin.
